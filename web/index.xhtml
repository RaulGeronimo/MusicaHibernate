Acerca del Proyecto

1Insertar Registros

Este sitio web va a registrar datos en una base de Datos que contiene 5 tablas, los cuales son: Album, Artista, Canciones, Disquera y Grupo.

2Mostrar Listas

Este sitio web mostrara todos los registros insertados en la Base de Datos en forma de lista, asimismo se podrán realizar cambios en los registros de cada una de las tablas, o en su defecto se eliminará el Registro. También se podrá realizar la búsqueda de los registros, así como la exportación en distintos formatos, los cuales son: Excel, PDF, CSV.
